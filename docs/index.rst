PyRth documentation
====================

PyRth is a Python package for analyzing thermal transient measurements using Network Identification by Deconvolution (NID) methods. It enables thermal characterization of electronic components and systems through time constant spectrum and structure function analysis.

The documentation provides a comprehensive guide to using the PyRth package, including installation instructions, usage examples, and detailed API references. For those interested in the mathematical foundations, the theory section explains the algorithms and methods implemented in PyRth, including deconvolution methods, network construction, structure function algorithms, and optimization techniques.
