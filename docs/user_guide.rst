User Guide
==========

This section provides practical step-by-step tutorials and advanced workflows for PyRth.

**Getting Started** - Start here if you're new to PyRth. This section covers basic concepts, installation verification, and your first thermal transient analysis.

**Usage** - Once you're familiar with the basics, explore the different evaluation methods, data input modes, and processing workflows available in PyRth.

**Troubleshooting** - Reference this section when you encounter issues during analysis. It provides solutions for common problems with impedance calculation, deconvolution, optimization, and structure function analysis.
